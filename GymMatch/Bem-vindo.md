## Resumo executivo

GymMatch resolve um problema simples: pessoas treinam lado a lado todos os dias na academia e nunca se falam. A plataforma usa um sistema de **QR code por academia** para criar comunidades fechadas — você só vê e interage com quem treina no mesmo lugar que você. O match só acontece quando é mútuo, eliminando o constrangimento de uma abordagem direta.

O produto combina a mecânica de swipe do Tinder com o contexto fitness, adiciona um chat em tempo real e uma IA assistente (Lucia) para manter os usuários engajados. A monetização é via planos de assinatura com limites progressivos.
